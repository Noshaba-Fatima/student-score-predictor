# 🎓 Student Exam Score Predictor (ML Web App)

A machine learning-powered web application built using **Streamlit** that predicts student exam scores based on academic and lifestyle factors.

---

## 🚀 Live Demo
👉 https://student-score-predictor.streamlit.app (add your link here)

---

## 📌 Project Overview

This project predicts a student's exam score using inputs like:
- Study hours
- Attendance
- Sleep hours
- Internet access
- Study method
- Course type

It also provides:
- 📊 Insights based on student behavior
- 📈 Visual comparison charts

---

## 🧠 Machine Learning Model

- Model trained using **Scikit-learn**
- Preprocessing includes **OneHotEncoding**
- Saved using **Joblib**
- Deployed using **Streamlit Cloud**

---

## ⚙️ How It Works

1. User enters student details
2. Model processes input data
3. Prediction is generated
4. Insights are displayed
5. Visualization shows performance comparison

---

## ☁️ Model Deployment

Since the model file is large, it is stored on **Google Drive** and dynamically loaded in Streamlit using `gdown`.
